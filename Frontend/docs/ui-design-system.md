# UI Design System
- modern
- premium
- clean
- fitness-oriented
- professional SaaS

Avoid generic admin templates.

---

## Visual Style

### Layout
- spacious
- clean grids
- consistent spacing
- information-focused

### Cards
- rounded-2xl
- subtle borders
- glassmorphism allowed
- soft shadows

### Buttons
- rounded-xl
- smooth hover transitions
- avoid harsh effects

---

## Typography

### Font
- Inter

### Heading Style
- semibold
- clean hierarchy

### Text Rules
- avoid oversized text
- prioritize readability

---

## Animations

Use:
- Framer Motion
- subtle transitions
- scale hover effects
- smooth page transitions

Avoid:
- excessive bounce
- distracting animations

---

## Dashboard Feel

The dashboard should resemble:
- modern SaaS platforms
- premium analytics dashboards
- gym management systems
- high-end admin panels

NOT old-school enterprise dashboards.
