## Core Rules

---

## UI Rules

- Avoid generic dashboard UI
- Use modern fitness SaaS aesthetic
- Use clean spacing
- Use rounded-2xl cards
- Use subtle shadows only
- Use premium dark UI
- Use smooth animations
- Use responsive layouts

---

## Code Rules

- Avoid duplicated code
- Avoid giant components
- Prefer reusable hooks
- Avoid inline styles
- Avoid hardcoded values
- Keep components modular
- Separate UI and business logic

---

## Do NOT Use

- Bootstrap
- Material UI
- jQuery
- Redux unless necessary
- Plain CSS files unless required
- Random colors
- Generic templates
